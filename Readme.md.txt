App Send Mail

O objetivo por trás do desenvolvimento deste app é colocar em prática conhecimentos sobre orientação a objetos em PHP e o uso de recursos como namespaces e tratamento de erros com o try... catch. Também está sendo utilizada a biblioteca PHPMailer.

Esta será uma aplicação funcional, capaz de enviar e-mails através do serviço de SMTP do gmail.

Este projeto faz parte de um curso de desenvolvimento web criado pelos instrutores Jorge Sant Ana e Jamilton Damasceno.
